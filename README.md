# Java Sorting Algorithms

Невелика демонстрація двох класичних алгоритмів сортування:

-  **Insertion Sort** — сортування вставками  
-  **Selection Sort** — сортування вибором  

Обидві програми:
- виводять процес сортування по кроках українською;
- показують стан масиву після кожної ітерації;
- допомагають зрозуміти логіку роботи алгоритму.
